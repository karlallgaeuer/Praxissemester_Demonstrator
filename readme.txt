1. Install Git
2. Install Java (1.8 or more) JDK 
3. Import project from Git (https://git.science.uu.nl/v.kasalica/biotoolcomposedemo)
4. Install Node.js (https://nodejs.org/en/download/)
5. Install Maven (https://maven.apache.org/install.html)
6. Fetch the dependencies with Maven
7. Run Application.java (Starts the backend with Spring Boot (for the REST-API))
8. In the command line, go into the directory "/biotoolcompesedemo/src/main/webapp/js" by typing "cd [directory]"
9. To start the Node.js server type "node server.js" (Port changable by editing server.js) (Default port 8080)
10. Open "http://localhost:8080/" in your browser
